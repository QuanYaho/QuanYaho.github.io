javaScript中的坐标属性

【1】在文档（document）对象里面用：

(1)

scrollWidth        
scrollHeight

获取对象的滚动宽度(滚动条可以滚动的宽度，相当于整个页面的总宽度的样子–网页正文全宽)

(2)

scrollLeft        
scrollTop

设置或获取位于对象左边界和窗口中目前可见内容的最左端之间的距离（页面利用滚动条滚动到右边时，隐藏在滚动条左边的页面宽度—页面被卷去左边）

例子：
document.body.scrollHeight

【2】在节点对象里面用：

(3)

offsetLeft
offsetTop

没有定位时，获取对象相对于版面（根元素）;
有定位时，获取当前对象相对与offsetParent元素的距离

注意：



offsetLeft.left 指当前对象到其上级层左边的距离。

不能对其进行赋值.

设置对象到其上级层左边的距离请用style.left属性。

  • 区别：

    • style.left返回值除了数字外还带有单位px；
    • 如对象到其上级层左边的距离设定值为百分比，style.left返回此百分比,而offsetLeft则返回到其上级层左边的距离的值；
    • 如果没有给 HTML 元素指定过 left样式，则 style.left返回的是空字符串；

>

(4) 

offsetWidth
offsetHeight

获取当前对象的宽度。

【区别】

style.width也是当前对象的宽度(width+padding+border)

(5)

clientWidth
clientHeight

获取对象可见内容的宽度，不包括滚动条，不包括边框；

(6)

clientLeft
clientTop

获取对象的border宽度

例子：

box.offsetWidth

box.offsetLeft

【3】事件里面用的：
event.clientX
event.clientY   // 相对文档的垂直座标

event.offsetX  // 相对容器的水平坐标
event.offsetY  // 相对容器的垂直坐标
【4】屏幕的：
window.screenTop 网页正文部分上
window.screenLeft 网页正文部分左

window.screen.height  屏幕分辨率的高
window.screen.left  屏幕分辨率的宽

window.screen.availHeight  屏幕可用工作区的高度
window.screen.availWidth  屏幕可用工作区的宽度



document.documentElement.scrollTop 垂直方向滚动的值
